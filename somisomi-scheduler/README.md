# SomiSomi Scheduler

Employee scheduling tool for SomiSomi The Woodlands.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

## Features

- **Employees Tab** — Manage 15 employees with roles, unavailability, tags, and trainee tracking
- **Time-Off Tab** — One-time time-off requests (separate from recurring unavailability)
- **Rules Tab** — Configurable scheduling rules with toggles, adjustable values, drag-to-reorder priority lists, and employee-specific fixed rules
- **Coming Soon** — School calendar, auto-generate schedule builder

## Tech Stack

- React 18 + Vite
- localStorage for persistence
- No external dependencies
